# Uniweb

교육용 웹 게임 서비스 

---

## 목적
- 높은 접근성을 가진 게임을 활용하여 아이들의 자연스러운 학습효과를 기대 

---

## 주요 기능
1. **게임 생성**  
   - 사용자가 원하는 문제와 정답을 입력하여 게임 내 문제 생성
2. **게임 공유**  
   - 본인이 만든 문제와 정답을 공유할 수 있고 타인의 문제와 정답도 가져올 수 있어 다양한 문제 풀이 가능
